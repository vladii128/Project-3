# Supermarket Sales Dashboard in Excel 

## Overview
This project is an interactive Excel dashboard created for a Big Data Analytics home assignment.  
The goal is to analyze supermarket sales data and present useful business insights through Pivot Tables, Pivot Charts, and Slicers.

## Objective
The dashboard is designed to help explore:

- customer behavior
- sales performance
- revenue trends
- time-based patterns
- top-selling products and categories

## Dataset
The dataset contains supermarket sales data with fields such as:

- Customer_ID
- Gender
- Marital_Status
- Homeowner
- Children
- Annual_Income
- City
- State_or_Province
- Country
- Product_Family
- Product_Department
- Product_Category
- Units_Sold
- Revenue

This dashboard focuses only on **2012–2013** sales data.

## Dashboard Features
The Excel dashboard includes:

- Pivot Tables
- Pivot Charts
- Slicers for interactive filtering
- customer segmentation analysis
- sales and revenue analysis
- time-series trend analysis
- top-selling product/category analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard formatting tools
